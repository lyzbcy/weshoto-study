# 急速备考学习追踪网站 · 本地预览说明

## 文件说明

- `一键启动前端.bat`：启动本地预览服务并自动打开浏览器
- `一键关闭前端.bat`：关闭本地预览服务并释放端口
- `local-preview-server.js`：静态文件服务 + 进度数据读写 API（端口 8091）

## 使用方式

1. 双击 `一键启动前端.bat`
2. 脚本会启动服务、等待首页 200 自检通过后自动打开浏览器
3. 访问 `http://127.0.0.1:8091/`
4. 在网页上点掌握度按钮、写笔记，会实时保存到 `progress.json`
5. 需要停止时，双击 `一键关闭前端.bat`

## 数据文件

- **`progress.json`**：所有学习进度数据（掌握度、每日时长、笔记）
- 网页上的所有操作会通过 `/api/save`、`/api/daily` 实时写回此文件
- 每次写入前会自动备份到 `progress.json.bak`
- 也可手动编辑此文件，网页每 30 秒轮询刷新

## API 接口

| 接口 | 方法 | 作用 |
|---|---|---|
| `/api/progress` | GET | 返回完整 progress.json |
| `/api/save` | POST | 保存某模块的掌握度/时长/笔记 |
| `/api/daily` | POST | 保存某天某考试的总时长/笔记 |

## 解决的问题

1. 纯前端无法读写本地文件，故用 Node 预览服务器提供读写 API
2. 启动脚本做了首页健康检查，避免「服务没起来就开浏览器」的空页问题
3. 关闭脚本按 `local-preview-server.js` + 项目路径双重特征定位进程，不会误杀其他 node 服务

## 备注

- 启动脚本使用 UTF-8 代码页（`chcp 65001`），中文路径不乱码
- 端口固定 8091，如被占用请先运行关闭脚本或手动修改 `local-preview-server.js`
- 操作系统考试相关内容仅在「操作系统考试」Tab 内显示，与企微考试数据完全隔离
