## 二、5场景核心功能·官方精确定义（实操演示务必用这个口径）

### 🎯 活码家族（场景一二三四五通用）

| 活码 | 分流依据 | 官方定义 |
|---|---|---|
| **员工活码** | 按员工 | 一码绑单个/多个员工，客户扫码添加员工为好友，**自动打标签+发欢迎语**，可设推广渠道、统计获客效果。支持单人/多人两种 |
| **客户群活码** | 按群 | 支持**顺序入群**和**分类进群**，群满后**自动创建新群**并自动切换承接。支持自动打标签+欢迎语 |
| **区域活码** | **LBS地理位置** | 自动识别客户实时地理位置，扫码加**就近门店的导购员/门店群**。品牌运营/门店店长可创建多个，适用线下门店拉新。**支持批量导入门店信息** |
| **识客码** | 按新老客身份 | **自动识别新老客户**：新客扫码加好友，老客扫码**直接进小程序/H5**（无需再加好友）。适用新老客差异化权益、点餐、一码多用 |
| **识客群码** | 按标签 | 满足客户分层进群：新客扫码入群，老客直接展示小程序/H5；按标签入群（有标签进A群，无标签进B群） |
| **群短链** | 渠道 | 活码可生成短链接，放短信/官网/广告落地页，客户点击拉起微信直接入群 |

> **区域活码 vs 员工活码（官方口径）**：区域活码适合「一张海报物料多门店共用」，可长期使用，门店增减随时调整；员工活码适合「专店专人专用」，**不支持按地理位置推荐**。

### 📤 群发四件套（场景一二三必背）

| 类型 | 官方定义 | 频次限制 |
|---|---|---|
| **群发客户** | 管理员建任务选客户→通知员工→员工确认后群发。**不同员工向同一客户每天可群发1条** | 每日1条共用 |
| **群发客户群** | 管理员建任务选群→通知群主→群主确认后群发。**群主向同一客户群每天最多1条** | 每日1条共用 |
| **群发朋友圈** | 管理员统一下发→员工确认发布到客户朋友圈 | 每天3条共用 |
| **超级群发** ⭐ | 管理员配置任务，指定**群发客户群+发送人+发送内容+起止时间**，由**多个客户群成员**在聊天工具栏查看并执行群发客户群任务，**突破每日群发次数限制**，对客户群多次触达 | 突破次数限制 |

> **超级群发三大官方特点（实操加分项）**：
> 1. 理论上**一天内多次触达**客户群（普通群发每天1条，超级群发突破此限制）
> 2. 适用社群多次触达：产品上新提醒、训练营开课/督学提醒、社群运营SOP定时多次群发
> 3. **每个超级群发任务最多指定100名员工执行**；可指定发送人（多人群随机选一名），未指定则由群主发
> 4. **前提**：必须先配置「客户待办」聊天工具栏，员工通过工具栏任务提示完成

### 💬 标准化营销（SOP，场景三四五通用）

**官方定位**：自动化营销引擎。管理员配置规则，**客户触发条件后自动按规则推送标准化内容**。
- 典型场景：加客户后第1天发什么、第2天发什么（分时段自动推送）
- 三大触发类型：**群发客户、群发客户群、跟进客户**
- 管家独有，企微原生不支持

### 📞 微信客服（场景三）

**官方定位**：客户**无需加好友**即可临时咨询，降低服务门槛。
- 一个客服账号可创建**多渠道**（公众号/视频号/小程序/客户单聊群聊）
- 管家比原生增强：接待组管理、满意度评价、超时提醒、排队提示语、**自助导航**
- **不需加好友、不依赖会话存档**

## 三、会话内容存档（场景四五六 + 笔试高频）

### 三个版本（官方口径，必背）

| 版本 | 存档范围 | 关键差异 |
|---|---|---|
| **办公版** | 仅内部沟通 | 内部单聊/群聊 |
| **服务版** | 内部 + 外部服务沟通质量管理 | 员工与客户间（原版本） |
| **企业版** | 内外部服务沟通 + **VOIP语音通话** | 含语音通话，最全 |

- **可试用30天**；3个版本**支持的功能相同**，区别仅在存档范围
- 支持消息类型：文本、图片、文件（含视频）、语音条、撤回消息；**语音通话VOIP仅企业版**

### 开通链路（实操口述必背）

```
企微原生后台【安全与管理→管理工具→会话内容存档】开启（提供确认函+配置参数）
    ↓ 参数配到
企微管家「我的企业→系统管理→配置管理」
    ↓
管家用参数做规则/AI分析
```
> 口诀：「**企微开门给钥匙 → 管家用钥匙看**」。会话存档参数在「我的企业→系统管理→配置管理」，**不在会话存档菜单里**。

### 会话存档五大能力

| 能力 | 说明 |
|---|---|
| **内容检索** | 按员工/客户/文件类型/时间/关键词检索；语音智能转文字 |
| **内容审计** | 敏感词/敏感行为实时预警；可加白名单（财务/高管聊天仅超管可查） |
| **服务质量** | **超时回复提醒**：员工超时未回复自动提醒（最低触发5分钟） |
| **情感分析** | AI识别员工/客户情感倾向 |
| **统计分析** | 单聊/群聊/会话类别/存档开启情况全维度统计 |

### 敏感词预警 + 超时回复（场景四五，实操原题）

| 功能 | 路径 | 关键参数 |
|---|---|---|
| **敏感词预警** | 会话存档→内容审计→敏感词预警 | 自定义批量导入 或 行业词库（金融/IT/教育）；触发动作：拦截/告警 |
| **超时回复** | 会话存档→服务质量→超时回复规则 | **最低触发5分钟**；可设工作日/休息日/项目不同规则；可设机器人+白名单 |

> **依赖前提**：敏感词预警和超时回复都**依赖会话存档**，必须先开通会话存档 license。

## 四、AI功能家族（5场景都涉及）

| AI功能 | 看什么 | 实时/T+1 | 所在菜单 |
|---|---|---|---|
| 智能问答（员工智能助手） | 基于知识库生成话术，侧边栏一键发送 | 实时 | 客户营销→AI智能体 |
| 话题洞察 | 群聊话题趋势+情感（产品价格/库存/咨询/评价/投诉） | T+1 | 会话存档→会话智能 |
| 客户洞察 | 单客户意向/异议/痛点+跟进建议 | 实时 | 会话存档→会话智能 |
| AI服务质检 | 员工服务规范（AI+规则全量检测，替代人工抽检） | 每日批量 | 会话存档→会话智能 |
| AI分析智能体 | 多模型对比+文件/多模态分析 | 实时 | 客户营销→AI智能体 |
| AI知识库 | 上传文档/FAQ，员工侧边栏智能问答 | 实时 | 客户营销→AI智能体 |

> **AI在管家后台分两处，别记串**：
> - **客户营销→AI智能体**（3项）：AI聊天智能体、AI分析智能体、AI知识库
> - **会话存档→会话智能**（3项）：话题洞察、AI合规审查、AI服务质检

## 五、版本支持（笔试「功能归属」用）

企微管家版本：**标准版 / 高级版 / AI版**

| 功能 | 标准版 | 高级版 | AI版 |
|---|---|---|---|
| 员工活码、识客码、客户群活码、区域活码、识客群码、群短链 | √ | √ | √ |
| 好友裂变、任务裂变、群裂变、红包裂变 | √（红包需充值） | √ | √ |
| 群发客户、群发客户群、群发朋友圈、超级群发 | √ | √ | √ |
| 一键拉群（拉群模板） | × | √ | √ |
| 线索、微信线索、广告管家、线索设置 | × | √ | √ |
| 智能外呼 | 按需付费 | 按需付费 | 按需付费 |

> **判别口诀**：基础获客/群发/裂变 = 全版本都有；线索管理/广告管家/一键拉群 = **高级版起**；智能外呼 = **单独付费**。
